# 서비스 기획서: 영화 선착순 할인 이벤트 통합 대시보드 (가칭: MovieEventCalender)

## 1. 서비스 개요

* **목적:** CGV(스피드쿠폰), 메가박스(빵원티켓), 롯데시네마(무비싸다구) 3사의 선착순 영화 할인 이벤트 정보를 단일 웹페이지로 제공함.
* **핵심 가치:** 사용자가 이벤트 일정을 사전에 파악하고 대비하도록 외부 캘린더(Google Calendar, Apple Calendar 등) 연동을 지원해, 별도 자체 알림(Push) 서버를 구축하지 않고도 알림 환경을 구성함.
* **타겟 고객:** 영화 관람 빈도가 높고 혜택 정보를 적극적으로 탐색하는 체리피커와 영화 마니아.
* **플랫폼:** 모바일 반응형 웹 (Responsive Web), 앱 설치 불필요.

## 2. 기획 배경 및 문제 정의

* **정보의 파편화:** 3사 영화관 앱을 개별적으로 실행해 숨겨진 이벤트 페이지를 확인해야 하는 번거로움이 있음.
* **일정 관리의 부재:** 기존 커뮤니티나 앱은 단발성 알림에 그치고, 사용자가 다가오는 여러 영화의 예매 일정을 한눈에 파악해 본인의 일정에 미리 등록해 두는 기능이 취약함.

## 3. 핵심 컨셉 (Zero-Server Cost 전략)

* 자체 알림(Push) 서버(DB, 백엔드 로직, FCM 등)를 구축하지 않고, 사용자의 기기에 내장된 캘린더 앱을 활용해 서버 유지 비용을 0원으로 수렴하는 정적 아키텍처.

## 4. MVP 주요 기능

### 4.1. 3사 통합 타임라인 대시보드

* 현재 진행 중이거나 예정된 3사 할인 이벤트 목록을 시간순으로 정렬해 제공함.
* 표시 정보: 극장명, 영화 제목, 이벤트 종류, 티켓 오픈 일시, 잔여 시간(D-Day/시간), 해당 극장 예매 링크.

### 4.2. 캘린더 사전 연동 (사전 대비 목적)

사용자가 이벤트 발생 며칠 전부터 일정을 파악하고 대비하도록 두 가지 방식으로 연동을 제공함.

1. **단건 일정 추가 (Template URL / 권장):**
   * 사용자가 특정 이벤트의 [내 캘린더에 추가] 버튼을 클릭하면 Google Calendar 생성 URL(`https://calendar.google.com/calendar/render?...`)로 이동함.
   * 사용자의 캘린더에 즉시 등록되며, 티켓 오픈 시간 기준 10분 전에 알림을 설정함.
   * **장점:** 게릴라성(1~2일 전) 공지 이벤트도 사용자가 발견하는 즉시 지연 없이 캘린더에 반영할 수 있음.
2. **전체 일정 구독 (ICS URL 제공 / 보조):**
   * 대시보드에서 제공하는 `.ics` URL을 사용자가 본인 캘린더에 '구독' 형식으로 추가함.
   * **단점:** 캘린더 앱 자체의 동기화 주기(12~48시간)로 인해 단기 공지 이벤트가 누락될 수 있음.

## 5. 시스템 아키텍처 및 기술 스택

* **Data Collection (크롤링 파이프라인):**
  * **도구:** Python (BeautifulSoup, Selenium)
  * **실행 환경:** GitHub Actions (스케줄러 기반 자동화, 무료)
  * **로직:** 3사 이벤트 페이지 파싱 -> 이벤트 데이터를 정적 JSON 및 ICS 파일로 변환 -> GitHub 리포지토리(Repository)에 배포함.
* **Front-end (정적 웹사이트):**
  * **도구:** React, Vue 또는 Vanilla JS + CSS Framework (Tailwind 등)
  * **호스팅:** GitHub Pages, Vercel, Netlify (무료 티어 활용)
  * **로직:** 정적 JSON 데이터를 가져와(Fetch) 사용자 브라우저에서 목록을 렌더링함.

## 6. 현실적 리스크 및 한계점

1. **동적 페이지 크롤링의 취약성:**
   * 3사 웹사이트는 DOM 구조가 수시로 변경되며, 특정 극장의 경우 봇 접근 차단(Anti-bot) 솔루션을 적용할 가능성이 큼. GitHub Actions의 IP 대역이 차단될 경우 프록시 우회 등의 추가 조치가 필요하며, 이는 비용을 초래할 수 있음.
2. **선착순 이벤트 공지 시점의 불규칙성:**
   * "며칠 전 미리 알림"이라는 기획 의도와 달리, 극장 측에서 티켓 오픈 당일이나 하루 전날 이벤트를 갑자기 공개할 때가 있음.
   * 이 경우 ICS 캘린더 구독 방식은 동기화 지연으로 소용이 없어지므로, 대시보드 자체를 주기적으로 방문하도록 유도하거나 단건 캘린더 추가 기능을 메인으로 내세워야 함.
