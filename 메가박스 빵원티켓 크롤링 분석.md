# **메가박스 '빵원티켓' 크롤링 기술 분석 및 구현 전략**

## **1\. 데이터 수집 엔드포인트**

* **URL**: https://m.megabox.co.kr/on/oh/ohe/Event/eventMngDiv.do  
* **Method**: POST  
* **특징**: RESTful API처럼 JSON을 반환하는 것이 아니라, 클라이언트가 요청한 파라미터에 맞춰 서버에서 HTML 태그를 생성한 뒤 문자열 형태로 반환함.

## **2\. Request Payload (검색 파라미터)**

POST 요청 시 x-www-form-urlencoded 형식으로 아래 데이터를 전송함. eventTitle 필드를 통해 서버 단에서 필터링을 수행할 수 있음.

```json
{  
    "totCnt": 1,  
    "currentPage": "1",  
    "recordCountPerPage": "10",  
    "eventTitle": "빵원",  // 핵심 타겟팅 파라미터  
    "eventDivCd": "",  
    "eventTyCd": "",  
    "eventStatCd": "ONG",  // 진행중(Ongoing)인 이벤트  
    "orderReqCd": "ONGlist"  
}
```

## **3\. Response 파싱 및 데이터 추출 로직**

응답으로 돌아오는 HTML 구조를 분석하여 데이터를 추출함.

* **이벤트 아이디(Event ID) 추출**:  
  \<a href="javascript:void(0)" onclick="javascript:fn\_eventDetail('20141', 'Y', 'N');"\>  
  위 태그에서 정규표현식(fn\_eventDetail\\('(\\d+)')을 사용하여 첫 번째 인자값(20141)을 추출해야 함.  
* **텍스트 노드 추출**:  
  p.title 클래스에서 이벤트 명칭 추출.  
  p.date 클래스에서 이벤트 기간 추출.  
* **최종 예매 URL 조합**:  
  추출한 Event ID를 결합하여 최종 접근 가능한 URL 생성.  
  https://megabox.co.kr/event/detail?eventNo={Event\_ID}

## **4\. Python 크롤링 구현 코드**

```python
import requests  
from bs4 import BeautifulSoup  
import re

def get\_megabox\_zero\_tickets():  
    url \= "\[https://m.megabox.co.kr/on/oh/ohe/Event/eventMngDiv.do\](https://m.megabox.co.kr/on/oh/ohe/Event/eventMngDiv.do)"  
      
    \# 폼 데이터 페이로드  
    payload \= {  
        "currentPage": "1",  
        "recordCountPerPage": "10",  
        "eventTitle": "빵원",  
        "eventStatCd": "ONG",  
        "orderReqCd": "ONGlist"  
    }  
      
    try:  
        \# JSON이 아닌 form-data 전송 (data=payload)  
        response \= requests.post(url, data=payload)  
        response.raise\_for\_status()  
          
        soup \= BeautifulSoup(response.text, 'html.parser')  
        items \= soup.select('.event-list .item')  
          
        results \= \[\]  
        for item in items:  
            a\_tag \= item.find('a')  
            if not a\_tag:   
                continue  
              
            \# 1\. 이벤트 ID 추출 (정규표현식)  
            onclick\_text \= a\_tag.get('onclick', '')  
            match \= re.search(r"fn\_eventDetail\\('(\\d+)'", onclick\_text)  
            event\_id \= match.group(1) if match else None  
              
            \# 2\. 제목 및 날짜 추출  
            title\_tag \= item.select\_one('.title')  
            title \= title\_tag.text.strip() if title\_tag else "제목 없음"  
              
            date\_tag \= item.select\_one('.date')  
            date \= date\_tag.text.strip() if date\_tag else "날짜 없음"  
              
            if event\_id:  
                results.append({  
                    "title": title,  
                    "date": date,  
                    "event\_id": event\_id,  
                    "url": f"\[https://megabox.co.kr/event/detail?eventNo=\](https://megabox.co.kr/event/detail?eventNo=){event\_id}"  
                })  
                  
        return results

    except Exception as e:  
        print(f"Error fetching Megabox events: {e}")  
        return \[\]
```
