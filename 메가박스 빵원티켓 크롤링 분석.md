# 메가박스 '빵원티켓' 크롤링 기술 분석 및 구현 전략

## 1. 데이터 수집 엔드포인트

* **URL**: https://m.megabox.co.kr/on/oh/ohe/Event/eventMngDiv.do  
* **Method**: POST  
* **특징**: RESTful API처럼 JSON을 반환하지 않고, 클라이언트가 요청한 매개변수에 맞춰 서버에서 HTML 태그를 생성한 후 문자열 형식으로 반환함.

## 2. Request Payload (검색 파라미터)

POST 요청 시 x-www-form-urlencoded 형식으로 다음 데이터를 전송함. eventTitle 필드로 서버 측에서 필터링을 수행할 수 있음.

```json
{  
    "totCnt": 1,  
    "currentPage": "1",  
    "recordCountPerPage": "10",  
    "eventTitle": "빵원",  // 핵심 타겟팅 파라미터  
    "eventDivCd": "",  
    "eventTyCd": "",  
    "eventStatCd": "ONG",  // 진행중(Ongoing)인 이벤트  
    "orderReqCd": "ONGlist"  
}
```

## 3. Response 파싱 및 데이터 추출 로직

응답받은 HTML 구조를 분석해 데이터를 추출함. 안정적인 수집을 위해 아래 세부 처리 로직이 구현됨.

* **네트워크 지연 및 타임아웃 대비 (재시도 로직)**:  
  서버 지연으로 인한 에러를 방지하기 위해 요청에 10초 타임아웃을 설정하고, 연결 실패 시 2초 대기 후 최대 3회까지 재시도(Retry)함.
* **이벤트 아이디(Event ID) 추출**:  
  `<a href="javascript:void(0)" onclick="javascript:fn_eventDetail('20141', 'Y', 'N');">`  
  위 태그에서 정규표현식 `fn_eventDetail\('(\d+)'`을 사용해 첫 번째 인자값(20141)을 추출함.
* **순수 영화 제목 정제**:  
  `p.title` 클래스에서 긁어온 원본 제목(예: `[프로젝트 헤일메리] 빵원티켓`)은 배너 문구가 섞여 있으므로, 정규표현식 `\[(.*?)\]`을 적용해 대괄호 내부의 순수 영화명만 추출함.
* **상세 페이지 2차 파싱을 통한 정확한 시간 추출**:  
  * 목록 화면의 `.date` 태그는 오픈 날짜(YYYY.MM.DD)만 제공하므로 정확한 예매 시작 시간(HH:mm)을 파악할 수 없음.
  * 이를 해결하기 위해 추출한 `Event ID`를 조합해 이벤트 상세 페이지(`https://megabox.co.kr/event/detail?eventNo={Event_ID}`)에 GET 요청을 보냄.
  * 상세 페이지 본문에서 정규표현식 `기간\s*(\d{4})\s*\.\s*(\d{1,2})\s*\.\s*(\d{1,2}).*?(\d{1,2}:\d{2})` 매칭을 통해 정확한 시작 일시(YYYY-MM-DD HH:MM:00)를 완성함. (상세 페이지 파싱 오류 발생 시 목록의 날짜와 00:00:00 값을 대체 데이터로 사용함)
* **이미지 URL 및 최종 경로 조립**:  
  * `img` 태그의 `data-src` 속성에서 썸네일 이미지 주소를 획득함.
  * 고유 식별자는 `mega-{Event_ID}` 형식으로 관리함.

## 4. Python 크롤링 구현 코드

```python
import requests  
from bs4 import BeautifulSoup  
import re
import time

def get_megabox_zero_tickets():  
    url = "https://m.megabox.co.kr/on/oh/ohe/Event/eventMngDiv.do"  
      
    # 폼 데이터 페이로드  
    payload = {  
        "currentPage": "1",  
        "recordCountPerPage": "10",  
        "eventTitle": "빵원",  
        "eventStatCd": "ONG",  
        "orderReqCd": "ONGlist"  
    }  
      
    headers = {  
        "User-Agent": "Mozilla/5.0 (Linux; Android 10; SM-G981B) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/80.0.3987.162 Mobile Safari/537.36"  
    }  
      
    results = []  
      
    try:  
        # 1. 목록 페이지 조회 (타임아웃 10초 설정 및 최대 3회 재시도)  
        response = None  
        for attempt in range(3):  
            try:  
                response = requests.post(url, data=payload, headers=headers, timeout=10)  
                response.raise_for_status()  
                break  
            except requests.RequestException as e:  
                print(f"[Megabox] 목록 조회 실패 (시도 {attempt + 1}/3): {e}")  
                if attempt < 2:  
                    time.sleep(2)  
                else:  
                    raise e  
          
        soup = BeautifulSoup(response.text, 'html.parser')  
        items = soup.select('.event-list .item')  
          
        for item in items:  
            a_tag = item.find('a')  
            if not a_tag:   
                continue  
              
            # 1. 이벤트 ID 추출 (정규표현식)  
            onclick_text = a_tag.get('onclick', '')  
            match = re.search(r"fn_eventDetail\('(\d+)'", onclick_text)  
            event_id = match.group(1) if match else None  
              
            # 2. 제목 추출 및 정제  
            title_tag = item.select_one('.title')  
            raw_title = title_tag.text.strip() if title_tag else "제목 없음"  
              
            clean_title = raw_title  
            title_match = re.search(r'\[(.*?)\]', raw_title)  
            if title_match:  
                clean_title = title_match.group(1).strip()  
              
            # 3. 날짜 추출 (기본값 설정)  
            date_tag = item.select_one('.date')  
            date_str = date_tag.text.strip() if date_tag else ""  
            start_date = ""  
              
            if "~" in date_str:  
                parts = date_str.split("~")  
                start_date = parts[0].strip().replace(".", "-") + " 00:00:00"  
            else:  
                start_date = date_str.replace(".", "-") + " 00:00:00"  
                  
            # 4. 이미지 주소 획득  
            img_tag = item.select_one('img')  
            img_url = img_tag.get('data-src', '') if img_tag else ""  
              
            if event_id:  
                detail_url = f"https://megabox.co.kr/event/detail?eventNo={event_id}"  
                  
                # 5. 상세 페이지 진입 및 정확한 시간 파싱  
                try:  
                    detail_res = requests.get(detail_url, headers=headers, timeout=5)  
                    time_match = re.search(r'기간\s*(\d{4})\s*\.\s*(\d{1,2})\s*\.\s*(\d{1,2}).*?(\d{1,2}:\d{2})', detail_res.text)  
                      
                    if time_match:  
                        year, month, day, time_str = time_match.groups()  
                        start_date = f"{year}-{int(month):02d}-{int(day):02d} {time_str}:00"  
                          
                except Exception as e:  
                    print(f"[Megabox] 상세 페이지 파싱 오류 (ID: {event_id}) - 1차 날짜로 대체합니다: {e}")  
                      
                results.append({  
                    "title": clean_title,  
                    "date": start_date,  
                    "event_id": event_id,  
                    "url": detail_url  
                })  
                  
        return results

    except Exception as e:  
        print(f"Error fetching Megabox events: {e}")  
        return []

# 테스트 실행  
if __name__ == "__main__":  
    bbangwon_events = get_megabox_zero_tickets()  
    for event in bbangwon_events:  
        print(f"제목: {event['title']}\n기간: {event['date']}\n링크: {event['url']}\n")  
```
