# **롯데시네마 '무비싸다구' 크롤링 기술 분석 및 구현 전략**

## **1\. 이벤트 페이지 접근 특성**

* **모바일 전용 환경**: 무비싸다구 이벤트 배너 및 상세 페이지는 모바일 웹 환경에서만 정상적으로 노출됨.  
* **리다이렉션 로직**: PC 브라우저의 User-Agent로 모바일 페이지 접근 시 일반 이벤트 목록 페이지(https://www.lottecinema.co.kr/NLCHS/Event)로 강제 리다이렉션 처리됨.  
* **타겟 엔드포인트**: 모바일 메인 페이지인 https://www.lottecinema.co.kr/NLCMW/가 데이터 수집의 진입점임.

## **2\. 기존 정적 파싱의 한계 (DOM 구조 분석)**

* **Hidden Action**: 배너 버튼의 HTML 태그가 \<a href="#"\> 구조로 작성되어 있음.  
* **이벤트 위임**: 클릭 이벤트가 JavaScript로 처리되므로, 단순 HTML DOM 파싱 도구(BeautifulSoup 등)로 배너의 \<a\> 태그를 찾아도 목적지 URL을 추출할 수 없음.  
* **가변적 파라미터**: 실제 이동하는 URL은 https://www.lottecinema.co.kr/NLCMW/Event/EventTemplateSpeedMulti?eventId={가변ID} 형태이며, 이 {가변ID}를 동적으로 알아내는 것이 핵심 과제임.

## **3\. 데이터 원천(Data Source) 분석 및 신규 API 활용 전략**

* **1단계 (MainEventID 추출)**: 모바일 메인 페이지(https://www.lottecinema.co.kr/NLCMW/) 반환 HTML 내 `<script>` 태그 안의 `GetMobileMainBottomItemsjsonData` 변수(혹은 기타 인라인 데이터) 등에서 메인 이벤트 ID(`MainEventID`, 예: 201210016922014)를 추출.
* **2단계 (상세 데이터 API 직접 호출)**: 추출한 `MainEventID`를 활용하여 롯데시네마 이벤트 상세 정보 제공 API(`https://www.lottecinema.co.kr/LCWS/Event/EventData.aspx`)에 POST 요청을 보내면, 영화 상세 정보 및 렌더링에 필요한 모든 데이터가 포함된 JSON 응답을 직접 획득할 수 있음.
  * **Payload 예시**: 
    ```json
    {
      "MethodName":"GetSpeedEventDetailMulti",
      "channelType":"MW",
      "osType":"A",
      "osVersion":"Mozilla/5.0 ... (모바일 User-Agent)",
      "EventID":"",
      "MainEventID":"201210016922014"
    }
    ```

## **4\. 최종 크롤링 파이프라인 (API 직접 호출 기반)**

위 분석을 바탕으로, 복잡한 HTML 파싱이나 브라우저 에뮬레이션을 최소화하면서 상세 데이터를 정확하게 추출하는 프로세스는 다음과 같음.

1. **HTTP 요청 위장 (1차)**: 헤더의 User-Agent를 모바일 기기(iPhone/Android)로 명시하여 `https://www.lottecinema.co.kr/NLCMW/` 호출.
2. **MainEventID 추출**: 정규표현식 또는 문자열 파싱을 통해 웹페이지 내 인라인 주입된 데이터 블록을 탐색하여 최신 `MainEventID`를 획득.
3. **API POST 요청 (2차)**: 획득한 `MainEventID`를 포함한 POST Payload를 구성하여 `https://www.lottecinema.co.kr/LCWS/Event/EventData.aspx`로 전송. (User-Agent 필수 설정)
4. **JSON 파싱 및 상세 데이터 추출**:
   * 반환된 JSON에서 `SpeedEventDetail` -> `ItemGroup` 배열, 그리고 그 하위의 각 `Items` 배열을 순회.
   * `MovieNm` (영화 제목), `ProgressStartTime` (이벤트 오픈 시간, 예: "16:00:00"), `CpnDownloadMaxCnt` (총 쿠폰 할당 수량), `SpeedCpnDownCnt` (현재 남은 쿠폰 수량), `ProgressStartDate` 등 대시보드 리스트 구성에 필요한 정확한 핵심 데이터를 즉시 수집.
5. **최종 URL 조립**: 최종 이벤트 접근 링크(예: `https://www.lottecinema.co.kr/NLCMW/Event/EventTemplateSpeedMulti?eventId={MainEventID}`)를 함께 생성하여 `MovieEvent` 인터페이스 규격에 맞는 최종 대시보드 데이터 포맷 완성.
